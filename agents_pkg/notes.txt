SpeechToObjDetectionNode:
- waits with input() until user types a duration in seconds
- then records audio from microphone for that duration
- saves the audio to a WAV file
- it calls the GeminiAgent server with the audio file path
- receives back the BBox of the detected object
- publishes the BBox to a ROS2 topic


ObjTrackerNode:
- when a BBox is received
- initializes the tracker
- in the camera_callback, the tracker is updated with each new frame
- the middle point of the BBox is calculated
- steering angle can be derived from the middle point of the BBox
     - use the aligned frame, so PP can be used (if dist < 0.2m: car should stop)
     - use the pixel based control (stopping should be solved somehow)
